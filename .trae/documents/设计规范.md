# TalkPilot 设计规范 v1.0

> 单一来源：所有色值、间距、圆角、阴影、字体均定义在 `src/shared/theme/tokens.ts`，组件中通过变量引用，禁止硬编码。

---

## 一、设计理念

**浅色科技感（Light Tech）**

- 背景用极浅黄绿 `bgBase`，与强调色同色系，整体和谐
- 强调色 `#C2EA45` 荧光绿，贯穿全局
- 半透明卡片 + 绿色光晕边框，营造层次感
- 脉冲扩散环、入场动效，体现 AI 实时感
- 浮动标签装饰（"AI-powered" / "Real-time"），呼应 Onboarding 语言

## 七、动效规范

| 场景 | 参数 |
|---|---|
| 页面入场 | `opacity: 0→1, translateY: 20→0, duration: 500ms / spring(damping:18, stiffness:120)` |
| 麦克风脉冲 | `scale: 1→1.07, duration: 1500ms, easing: inOut(ease), repeat: -1 reverse` |
| 脉冲扩散环（内） | `scale: 1→1.6, opacity: 0.7→0, duration: 2000ms, easing: out(ease), repeat: -1` |
| 脉冲扩散环（外） | `scale: 1→2.1, opacity: 0.4→0, duration: 2800ms, easing: out(ease), repeat: -1` |
| 颜色过渡（CTA） | `withTiming, duration: 400ms` |
| Dot 宽度动画 | `withSpring(damping:15, stiffness:200)` |

## 八、组件规范

### 卡片
- 背景：`palette.bgCard`（半透明）
- 边框：`palette.accentBorder`
- 圆角：`radii.lg` 或 `radii.xl`
- 阴影：`shadows.card`

### 主按钮（CTA）
- 背景：`palette.accent` 渐变到 `palette.accentGradientEnd`
- 文字：`palette.textOnAccent`
- 圆角：`radii.pill`

### Ghost 按钮
- 背景：`palette.bgGhostButton`
- 边框：`palette.accentBorder`
- 文字：`palette.textPrimary`

### Badge / 浮动标签
- 背景：`palette.accentMutedMid`
- 边框：`palette.accentBorderStrong`
- 文字：`palette.textAccent`
- 圆角：`radii.pill`

### 进度条
- 轨道：`palette.neutralTrack`
- 填充：`palette.accent` → `palette.accentGradientEnd`（渐变）
- 高度：5px，圆角：3px
